## Commands

**/powerfulstone**: Shows the help menu.

**/powerfulstone reload**: Reloads the configuration files.

**\[!] /powerfulstone menu**: Opens the main menu.

**\[!] /powerfulstone create \<stone name> \[display name]**: Starts a powerful stone creation process.

**\[!] /powerfulstone toggle-name-hide**: Toggles your name's hidden state.

## Permissions

**powerfulstone.main**: Lets you see the help menu.

**powerfulstone.reload**: Lets you reload the configuration files.

**powerfulstone.menu**: Lets you open the main menu.

**powerfulstone.create**: Lets you start a powerful stone creation process.

**powerfulstone.toggle-name-hide**: Lets you toggle your name's hidden state.
